# Python and Data Tools for Non-Programmers
## General information
### Final Python exam

### Instructions

In this section, you will find the files and instructions for a Python project that must be completed individually in order to obtain ECTS credits for this course.


**1 - Submission Deadline**

- Deadline: To be announced

- What to submit: one Jupyter Notebook (.ipynb) file containing your solution

- File name format:

     FirstName_LastName_ImmatriculationNumber.ipynb

- Submission method: send the file by email to albuquerque@uni-bayreuth.de

Note: Late submissions will get lower grades!


**2 - Project Presentation**

You will give a short presentation of your project:

- Duration: 10 minutes (± 2 minutes)

- Format: presentation directly from your submitted Jupyter Notebook

- Equipment: my laptop (you do not need to bring your own)

Presentation dates:

To be announced

Presentations will take place during the usual lecture times and rooms.
If necessary, presentations may continue on additional days.

After the presentation, there will be approximately 5 minutes of questions about:

- your project, and

- general Python concepts.
